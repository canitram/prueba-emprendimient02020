HTML y CSS para principiantes.

HTML y CSS son dos herramientas importantes para dar inicio en el mundo del desarrollo web. Por lo tanto, en este curso aprenderás todo sobre HTML y CSS, desde lo más básico hasta un nivel intermedio, logrando desarrollar tu primera página we responsiva, sin la necesidad de aplicar lirerías externas para CSS.

Estilos con SASS

SASS es una herramienta importante hoy en dia para todo desarrollador web, ya que permite crear estilos mas ordenados, eficientes y con menos lineas de código. Por ello, en este curso aprenderás todo lo necesario para crear hojas de estilos desde la sintaxis de SCSS, implementando técnicas propias de SASS.
